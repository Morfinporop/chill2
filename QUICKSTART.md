# ChillGram — Быстрый старт

## 1️⃣ Запуск сервера

```bash
cd server
npm install
npm start
```

Сервер запустится на **http://localhost:8080**

## 2️⃣ Запуск клиента

Откройте новый терминал:

```bash
npm install
npm run dev
```

Клиент откроется на **http://localhost:5173**

## 3️⃣ Первый вход

1. **Введите номер** — например `+7 999 123 45 67`
   - Появится флаг страны 🇷🇺
   - Интерфейс будет светлым

2. **Введите код** — любые 4 цифры (например `1234`)

3. **Регистрация:**
   - Выберите цвет аватара (12 вариантов)
   - Введите имя: `Иван Петров`
   - Введите юзернейм: `ivan_petrov` (мин. 5 символов)

4. **Готово!** 
   - Тема автоматически станет тёмной
   - Вы в ChillGram!

## 🔐 Админ вход

Номер админа **скрыт в коде** для безопасности.

При входе отображается как: **4455**
Код: **1234**

## 🎨 Особенности

- ✅ **Флаги стран** — автоматически для +7 🇷🇺, +1 🇺🇸, +380 🇺🇦 и т.д.
- ✅ **Умная тема** — светлая при входе, тёмная после
- ✅ **Синий акцент** — #3390EC как в Telegram
- ✅ **Номера скрыты** — показываются только последние 4 цифры
- ✅ **Цветные аватары** — вместо картинок
- ✅ **Чистый CSS** — никаких зависимостей

## 🚀 Деплой на Railway

1. Создайте проект на Railway.app
2. Подключите GitHub репозиторий
3. Root Directory: `server`
4. Railway автоматически определит Node.js
5. Деплой!

URL будет вида: `https://chillgram-production.up.railway.app`

## 📱 Функции

- Реальный WebSocket для онлайн статусов
- Отправка сообщений в реальном времени
- Индикатор "печатает..."
- Галочки прочтения ✓ и ✓✓
- Поиск по чатам
- Создание чатов
- Смена темы

## 💡 Советы

- Откройте в **2 окнах** браузера для тестирования чата
- Используйте разные номера для разных пользователей
- Номера телефонов скрыты — видны только юзернеймы
- При регистрации выберите уникальный юзернейм
