# ChillGram — Защищённый Мессенджер

Полнофункциональный клон Telegram с реальным WebSocket сервером, API и современным UI.

## Функции

✅ **Реальный WebSocket** — онлайн статусы, мгновенные сообщения, индикаторы набора текста  
✅ **Node.js API** — Express сервер на порту 8080  
✅ **Регистрация с флагами стран** — автоопределение страны по номеру 🇷🇺 🇺🇸 🇺🇦  
✅ **Цветные аватары** — 12 цветов на выбор  
✅ **Админ** — номер скрыт в коде, отображается как "4455"  
✅ **Умные темы** — белая при входе, тёмная после входа  
✅ **Синий акцент** — #3390EC как в Telegram  
✅ **Приватность** — номера телефонов скрыты по умолчанию  
✅ **Приватные чаты** — 1 на 1 общение  
✅ **Группы и каналы** — создание, управление  
✅ **Типизация** — индикатор "печатает..."  
✅ **Галочки прочтения** — ✓ и ✓✓  
✅ **SVG иконки** — чистый дизайн без emoji  
✅ **Чистый CSS** — без зависимостей от Tailwind  

## Запуск

### Сервер

```bash
cd server
npm install
npm start
```

Сервер запустится на `http://localhost:8080`

### Клиент

```bash
npm install
npm run dev
```

Клиент откроется на `http://localhost:5173`

## Вход

### Админ
- Номер: скрыт в коде сервера (защищён)
- Отображается как: `4455`
- Код: `1234`
- Имя: `Создатель`
- Права: полный доступ, создание ботов/каналов

### Обычный пользователь
1. Введите номер телефона (появится флаг страны 🇷🇺)
2. Введите любой код (4+ цифры)
3. Выберите цвет аватара
4. Введите имя и юзернейм (мин. 5 символов)
5. Готово! Тема автоматически станет тёмной

## Технологии

**Frontend:**
- React 19 + TypeScript
- Vite 7
- Чистый CSS (без Tailwind зависимостей)
- Zustand (стейт)
- WebSocket клиент
- SVG иконки

**Backend:**
- Node.js 22
- Express
- WebSocket Server (ws)
- In-memory storage
